Zem email systme is very simple email sending system to send emails to multiple clients.

== Description ==

Zem email systme is very simple email sending system user can save multiple email and then send a pre defined email template with few clicks.

== Installation ==

It has very simple interface and simple installation required just install zememailsystem and activate it, now it ready to use no further installation required.

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.2.4 or greater
